# tischkicker--pico-automatisation
Es wurde mit Hilfe eines Raspberry Pi Pico [Modellname] ein einfacher Tischkicker Automatsiert. Hier für wurde eine Lichtschranke in den Toren genutzt und die mechansische Toranzeige durch jeweils ein weiteren Pico [Modellnamen] mit einer RGB-Matrix ersetzt. Der Tsichkicker erkennt automatisch Tore und zählt diese hoch.
